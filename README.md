# AppLogger

logging with color

## Installation

Add this line to your application's Gemfile:

    gem 'app_logger', :git => 'git://github.com/rawhide/app_logger.git'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install app_logger

## Usage

logger.info "hogehoge"
